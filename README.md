# TSE Analytics - 2024

Projeto que visa apresentar dados referentes a candidaturas dos partidos para as eleições municipais brasileiras de 2024. 
Assim, é possível oferecer mais clareza à população a respeito dos dados, e com isso, utilizar dados reais para aprendizado.

## Fonte dos dados 

Tribunal Superior Eleitoral (TSE): [Acesse os dados aqui](https://dados.gov.br/dados/conjuntos-dados/candidatos-2024).
